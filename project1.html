@tailwindcss/typography

Beautiful typographic defaults for HTML you don't control.
The official Tailwind CSS Typography Plugin is a set of 'prose' classes you can use to add beautiful typographic defaults to any vanilla HTML you don't control, like HTML rendered from Markdown, or pulled from a CMS.
